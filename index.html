clu by denete

A bot when you need a clue.

This Clu-bot will take entries in JSON data files and return a random entry or find matching entries if an argument is provided. Entries include the keyword, definition, and a reference URL. Written in Perl.

Dependencies

Net::IRC Text::Wrap JSON::XS Data::Dumper
